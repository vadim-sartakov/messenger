# Messenger Application

Simple text messenger application built with React and Node.js

### Features
- Authentication (No password required, username is enough)
- Chat rooms
- Friends list
- Text messages
- Responsive layout

### Stack
- Webpack
- React
- Redux
- Redux Saga
- Material UI
- Jest
- Node.js
- Express
- MongoDB
- JWT
- GraphQL
- Websockets
