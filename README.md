# Messenger Demo Application

Messenger application

### Features
- Authentication (No sign up required, username is enough)
- Chat rooms
- Text messages

### Libraries used:

#### Web application
- webpack
- react
- react-router-dom
- redux
- redux-saga
- redux-persist
- formik
- @material-ui

#### Server
- node.js
- express
- mongoose
- JWT
